# House Price Prediction using Machine Learning

This project predicts house prices using Machine Learning techniques on a real-world housing dataset.

## Project Objective
To build a regression model that predicts house prices based on features like:
- Area
- Bedrooms
- Bathrooms
- Parking
- Furnishing status
- Other property amenities

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Models Used
- Linear Regression
- Random Forest Regression

## Evaluation Metrics
- MAE (Mean Absolute Error)
- RMSE (Root Mean Squared Error)
- R² Score

## Visualizations
- House Price Distribution Histogram
- Correlation Heatmap
- Actual vs Predicted Prices Scatter Plot

## Key Insights
- Larger houses and more bathrooms increase property price.
- Furnished homes and air conditioning attract more buyers.
- Data-driven pricing can help avoid overpricing or underpricing.
